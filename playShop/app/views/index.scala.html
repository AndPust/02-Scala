This site has two controllers: ProductController and CartController.
For GET and POST requests you can use the HTML forms below.
For PUT and DELETE requests you will have to use curl or software like Postman.
The data in the requests has to be url-encoded.

ProductController:

Delete a Product

In order to delete a Product execute the following curl command with appropriate id and check the ProductList:
curl -X DELETE http://localhost:9000/product?id=1

Update a Product

In order to update a Product execute the following curl command with appropriate id, name and colour. Then check the ProductList:
curl -X PUT --data-urlencode "id=1" --data-urlencode "name=Car" --data-urlencode "colour=Red" http://localhost:9000/product

Delete a Category

In order to delete a Category execute the following curl command with appropriate name and check the CategoryList:
curl -X DELETE http://localhost:9000/category?name=Furniture

Delete a product from Categories

In order to delete a Category execute the following curl command with appropriate id anc category name and check the CategoryList (if you don't provide the category name then the id will be removed from all categories):
curl -X DELETE --data-urlencode "name=Furniture" --data-urlencode "id=1" http://localhost:9000/category

Update a Category

In order to update a Category execute the following curl command with appropriate name and new_name. Then check the CategoryList:
curl -X PUT --data-urlencode "name=Furniture" --data-urlencode "new_name=Chairs" http://localhost:9000/category
