This site runs on three controllers: FishController, VeggieController and FruitController.
For GET and POST requests you can use the HTML forms below.
For PUT and DELETE requests you will have to use curl or software like Postman.
The data in the requests has to be url-encoded.

Delete a product

In order to delete a product execute the following curl command with appropriate id and check the VegList:
curl -X DELETE http://localhost:9000/product?id=1

Update a product

In order to update a product execute the following curl command with appropriate id, name and colour. Then check the ProductList:
curl -X PUT --data-urlencode "id=1" --data-urlencode "name=Armchair" --data-urlencode "colour=Red" http://localhost:9000/product
